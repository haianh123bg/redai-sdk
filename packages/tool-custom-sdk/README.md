# @redonvn/p03-erp-v1-tool-custom-sdk

SDK stateless phục vụ chuẩn hóa OpenAPI/Postman thành mô hình tool, kèm executor HTTP và các tiện ích liên quan.

## Cài đặt

```bash
npm install @redonvn/p03-erp-v1-tool-custom-sdk
```

## Build

```bash
npm run build
```

## Typecheck

```bash
npm run typecheck
```
